TM 행동 설정 도우미

이미지 분류 모델을 행동으로 연결하기

Teachable Machine 모델을 불러오고, 라벨마다 화면 행동을 설정해 보세요.

모델 연결, 라벨 확인, 행동 설정, 웹캠 테스트와 출력 결과 확인까지 한 화면에서 차례대로 진행할 수 있습니다.

1. 모델 연결 2. 라벨 확인 3. 행동 설정 4. 실행 테스트
1

모델 연결

Teachable Machine에서 모델을 업로드한 뒤 공유 링크를 붙여 넣으세요.

준비 완료. 모델 링크를 입력해 주세요.
2

라벨별 행동 설정

모델 라벨이 인식되었을 때 보여 줄 문구나 이미지를 정합니다.

모델을 불러오면 라벨별 설정 카드가 자동으로 만들어집니다.
3

실행 테스트

웹캠 앞에 대상을 보여 주고 예측 결과를 확인합니다.

웹캠 화면이 여기에 표시됩니다.

주의: 가능하면 얼굴이나 개인정보가 나오지 않도록 대상을 카메라에 보여 주세요. 예측은 브라우저 안에서만 실행됩니다.

4

현재 예측과 출력 결과

신뢰도 기준 이상일 때만 설정한 행동이 실행됩니다.

아직 예측 결과가 없습니다.

행동이 실행되면 결과가 여기에 나타납니다.

모델이 헷갈린다면?

데이터 수, 라벨의 차이, 조명, 카메라 각도를 점검해 보세요.
